## Full width image

横幅图片：3,000 × 1,500

Width must be at least 1,500 px.

Tip: 3,000 px is best for a high-resolution display on all devices.

## Gallery

横幅图片：3,000 × 1,500

This gallery supports between 3 to 8 images. Images must be at least 1,500 x 750 pixels.
