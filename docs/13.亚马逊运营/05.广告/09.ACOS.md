如何降低 ACOS？

```md
ACOS = 广告花费 / 广告销售额

     = 平均 CPC x 广告点击数 /（商品单价 x 广告销售数量)

     = 平均 CPC x 广告点击数 /（商品单价 x 广告点击数 x 广告转化率）

     = 平均 CPC /（商品单价 x 广告转化率）
```

结论：降低 ACOS 只能从这三个指标入手：

1. 降低平均 CPC
2. 提高平均单价
3. 提高转化率
