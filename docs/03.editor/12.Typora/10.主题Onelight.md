新下载了一个主题 Onelight 试用，挖掘一下好看的样式，剖析一下 CSS 样式代码，借我所用。

## 主题特点

1. 字体搭配：英文等宽字体 Cascadia Code 和中文「喵字果汁体」搭配起来还算大小合适
2. 源代码模式：超宽屏显示，等宽字体支持 Markdown 表格的排版对齐
3. 主题风格偏灵动活泼

## 美中不足

1. 中文字体「喵字果汁体」仅支持中文简体，暂不支持繁体字符
