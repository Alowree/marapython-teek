## Objective

```md
ACOS = 广告花费 / 广告销售额

     = 平均 CPC x 广告点击数 /（商品单价 x 广告销售数量)

     = 平均 CPC x 广告点击数 /（商品单价 x 广告点击数 x 广告转化率）

     = 平均 CPC /（商品单价 x 广告转化率）
```

Maximum affordable CPC for Charge Rise II - category

| CPC  | U/P   | ACOS   | CVR   |
| ---- | ----- | ------ | ----- |
| 0.50 | 39.99 | 33.16% | 3.77% |
| 0.60 | 39.99 | 39.80% | 3.77% |
| 0.70 | 39.99 | 46.43% | 3.77% |
| 0.80 | 39.99 | 53.06% | 3.77% |

Theoretically, given these three constraints, (Unit Price=39.99, CVR=3.77%, and ACOS<=40%), the average CPC of this ads group should never exceed $0.60.

In short, the day to day task is to decide what bids we should offer for each keyword/asin, so that the target CPC<=0.60 (hence the target ACOS<=40%) of each ads campaign is met.

## Final Naming Convention Summary

**Format:** `[ProductCode].[CampaignType].[Target].[OptionalSuffix]`

| Element          | Description                        | Examples                               |
| :--------------- | :--------------------------------- | :------------------------------------- |
| `ProductCode`    | Product identifier                 | `sfq-16`, `sfq-18`, `sfq-20`, `sfqold` |
| `CampaignType`   | Type of campaign                   | `Keyword`, `Category`, `ASIN`          |
| `Target`         | What the campaign targets          | `alarm-clock`, `clock-radios`, `2510`  |
| `OptionalSuffix` | Date code or version (YYMM format) | `2411` (Nov 2024), `2602` (Feb 2026)   |

## Useful Excel Formulas for Parsing These Names

Once your `Raw_Campaign_Data` table has these standardized names, you can use these formulas to extract components:

| To Extract         | Formula (assuming name in cell A2) |
| :----------------- | :--------------------------------- |
| Product Code       | `=TEXTBEFORE(A2, ".")`             |
| Campaign Type      | `=INDEX(TEXTSPLIT(A2, "."), 2)`    |
| Target             | `=INDEX(TEXTSPLIT(A2, "."), 3)`    |
| Suffix (if exists) | `=INDEX(TEXTSPLIT(A2, "."), 4)`    |

**Example:** For `sfq-16.Category.clock-radios.2602`:

- Product Code = `sfq-16`
- Campaign Type = `Category`
- Target = `clock-radios`
- Suffix = `2602`
